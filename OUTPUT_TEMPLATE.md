# 完整上架模式输出模板

使用 `ecommerce-product-listing-skill` 时，除非用户明确要求简化，否则按以下模板输出。

## 1. 产品定位

```text
1. 产品一句话定位：
2. 目标人群：
3. 核心使用场景：
4. 主要购买理由：
5. 上架卖点方向：
```

## 2. 标题建议

```text
1. 平台主标题：
2. SEO 长标题：
3. 简洁备选标题：
```

要求：

- 主标题和 SEO 长标题具备关键词累加感。
- 不能只是“品牌 + 型号 + 品类”。
- 不写未知参数。

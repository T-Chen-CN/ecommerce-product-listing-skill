# 跨境电商产品上架与本地化内容生产 Skill

这是一个可复用的 Agent Skill，用于根据产品信息、产品图片、目标国家和销售平台，生成适合直接投入运营使用的跨境电商上架内容。

## 适用场景

适用于：

- TikTok Shop / Shopee / Lazada / Amazon / Temu / Shopify 等平台
- 数码、家居、小家电、美妆个护、日用百货等消费类产品
- 标题、卖点、详情、SKU、关键词、规格、包装、主图 Prompt 等场景

## 本版本核心升级

### 1. 输出格式升级

Skill 会强制区分两类内容：

- **说明类内容**：正常文字输出
- **可直接复制的成品内容**：必须放进写作块

这样标题、详情、卖点、SKU、规格表、关键词、包装清单等内容可以直接复制使用，不会混入说明或备注。

### 2. 9 图模块升级

原本的“9 张产品图建议”已升级为：

- **9 个独立写作块**
- **每个写作块都是一个完整图片生成 Prompt**

每个 Prompt 都要求：

- 可单独复制使用
- 内容完整闭环
- 风格统一
- 产品一致
- 分工明确

### 3. 禁止 Prompt 互相引用

9 个 Prompt 之间不得出现以下表达：

- 保持与上一张图一致
- 延续前一张图风格
- 同前图
- 参考前图
- 与图1相同
- same as previous image

统一性必须通过在每个 Prompt 内部重新写清楚统一基线信息来实现，而不是相互引用。

### 4. 强化马来西亚市场本地化

当目标市场为马来西亚时：

- 以马来语为主
- 可自然保留常见英文电商关键词
- 文案短、自然、生活化
- 适合 TikTok Shop / Shopee / Lazada Malaysia 使用

## 使用方式

把整个文件夹放到你的 Agent / SubAgent / Skill 目录中，或直接把 `SKILL.md` 内容作为系统提示词 / Skill 说明导入。

## 推荐调用语句

```text
按照产品上架 Skill 处理这个产品。
目标国家：马来西亚
平台：TikTok Shop
产品信息如下：
……
按可直接复制格式输出。
标题、卖点、详情、SKU、规格、关键词、包装清单都用写作块。
9张产品图请输出成9个独立写作块，每个写作块都是一个单独可直接使用的图片生成 Prompt。
```

## 输出原则

- 常规说明内容正常输出
- 可直接复制使用的内容必须放进写作块
- 产品详情描述必须是正式可用成品
- 9图模块必须输出为 9 个独立 Prompt
- 9 个 Prompt 之间不得互相引用
- 不得编造未提供的参数、认证、授权或功效

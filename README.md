# 跨境电商产品上架与本地化内容生产 Skill

这是一个可复用的 Agent Skill，用于根据产品信息、产品图片、目标国家和销售平台，生成适合直接投入运营使用的跨境电商上架内容。

## 适用场景

适用于：

- TikTok Shop / Shopee / Lazada / Amazon / Temu / Shopify 等平台
- 数码、家居、小家电、美妆个护、服饰配件、日用百货、母婴玩具等消费类产品
- 标题、卖点、详情、SKU、关键词、规格、包装清单、风险提醒、9 张产品图提示词等内容生产场景

## 本版本核心优化

### 1. 标题从“短名称”升级为“电商搜索标题”

Skill 会要求标题采用关键词累加型结构，而不是简单的“品牌 + 型号 + 品类”。

标题会根据产品自动组合：

- 品牌或型号
- 核心品类
- 核心属性
- 主要功能
- 关键规格
- 使用场景
- 适用对象或适用需求
- 平台搜索关键词
- 目标市场本地化关键词

默认输出 3 个标题版本：

1. 平台主标题
2. SEO 长标题
3. 简洁备选标题

### 2. 9 图提示词从“纯产品摄影”升级为“电商信息图”

除非用户明确要求纯产品摄影，否则 9 张图默认都应是：

产品视觉 + 电商文字说明 + 清晰卖点 + 平台适配排版

每张图提示词都要求包含：

- 产品外观锁定
- 图片商业目的
- 画面构图
- 背景或场景
- 主标题
- 副标题
- 2-4 个卖点标签
- 可选小字说明
- 风格要求
- 禁止项

### 3. 9 图分工改为通用商业逻辑

本版本不绑定耳机、数码或任何单一类目，而是使用通用电商图逻辑：

1. 主图封面
2. 核心卖点图
3. 使用场景图
4. 结构材质图
5. 功能操作图
6. 参数规格图
7. 适配需求图
8. SKU 款式图
9. 包装清单或总结转化图

如果某个产品不适合某一张图，允许自动替换为对比说明图、尺寸比例图、使用前后场景图、卖点合集图、礼品套装图、适用空间图、细节放大图或安装步骤图。

### 4. 取消固定写作块和代码块要求

Skill 不再强制使用写作块或 Markdown 代码块。

输出只要求：

- 板块清晰
- 内容可直接复制
- 成品内容中不混入解释、备注、分析
- 用户指定格式时，优先按用户指定格式输出

### 5. 强化真实信息优先级

Skill 会要求：

- 参数表优先于图片
- 用户确认颜色优先于图片推测
- 未提供的信息不得编造
- 不得编造认证、防水等级、功效、官方授权、平台背书

## 推荐调用语句

```text
按照产品上架 Skill 处理这个产品。
目标国家：马来西亚
平台：TikTok Shop / Shopee / Lazada
产品信息如下：
……

请输出完整上架内容，并生成 9 张独立正方形产品图提示词。
标题要像真实电商平台标题，有关键词累加感。
9 图提示词要包含产品视觉和电商文字说明，不要只做纯产品摄影图。
```

## 输出原则

- 标题必须具有平台搜索标题的信息密度
- 9 图提示词必须具备电商信息图属性
- 所有提示词必须独立完整，不能互相引用
- 规则必须跨类目通用，不绑定单一产品类型
- 不得编造未提供的参数、认证、授权或功效

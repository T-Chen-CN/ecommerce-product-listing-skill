# 跨境电商产品上架与本地化内容生产 Skill

这是一个可复用的 Agent Skill，用于根据产品信息、产品图片、目标国家和销售平台，生成适合直接投入运营使用的跨境电商上架内容。

当前版本：**v2.0.0 Output Contract**

## 适用场景

适用于：

- TikTok Shop / Shopee / Lazada / Amazon / Temu / Shopify 等平台
- 数码、家居、小家电、美妆个护、服饰配件、日用百货、母婴玩具等消费类产品
- 标题、卖点、详情、SKU、关键词、规格、包装清单、风险提醒、9 张产品图提示词等内容生产场景

## v2.0 核心优化

### 1. 从“建议型 Skill”升级为“交付契约型 Skill”

v2.0 引入 Output Contract：

- 默认使用完整上架模式
- 固定 10 个输出板块
- 固定最低内容规模
- 固定质量门槛
- 低于门槛必须自动重写

目标是解决 Agent 产出不稳定、内容缩水、标题过短、9 图提示词过素的问题。

### 2. 标题强制具备电商搜索密度

默认输出 3 个标题：

1. 平台主标题
2. SEO 长标题
3. 简洁备选标题

主标题和 SEO 长标题必须具备关键词累加感，不能只是“品牌 + 型号 + 品类”。

### 3. 核心卖点固定 8 条

每条卖点必须包含：

- 卖点标题
- 用户利益说明
- 可用于详情页或图片文字的短文案

避免只输出“高品质、舒适、方便、耐用”等空泛内容。

### 4. 详情文案固定 7 个小节

详情文案必须包含：

1. 开头购买理由
2. 核心卖点
3. 使用场景
4. 适合人群
5. 规格参数
6. 包装清单
7. 购买提醒

详情文案必须是可直接复制使用的成品，不是分析说明。

### 5. 9 图提示词固定字段制

默认输出 9 张独立正方形 1:1 产品图提示词。

每张图必须包含：

- 图片目的
- 产品锁定
- 画面构图
- 背景/场景
- 主标题
- 副标题
- 卖点标签
- 小字说明
- 视觉风格
- Negative rules

9 图提示词禁止互相引用，禁止“同上”“参考上一张”“same as previous”。

### 6. 增加缩水输出判定

出现以下情况会被视为不合格：

- 默认板块缺失
- 标题少于 3 个
- 核心卖点少于 8 条
- 详情文案缺少 7 个小节
- 9 图提示词少于 9 张
- 任意图缺少主标题、副标题、卖点标签、小字说明或 Negative rules
- 关键词少于 30 个
- SKU 未覆盖全部用户提供款式
- 输出像摘要或提纲

不合格时，Agent 必须先重写，不能输出缩水版本。

## 文件说明

```text
ecommerce-product-listing-skill/
├── README.md
├── SKILL.md
├── QUALITY_GATE.md
├── OUTPUT_TEMPLATE.md
└── CHANGELOG.md
```

- `SKILL.md`：核心 Skill 规则
- `QUALITY_GATE.md`：质量门槛与缩水判定
- `OUTPUT_TEMPLATE.md`：完整上架模式输出模板
- `CHANGELOG.md`：版本迭代记录

## 推荐调用语句

```text
按照 ecommerce-product-listing-skill 处理这个产品。
目标国家：马来西亚
平台：TikTok Shop / Shopee / Lazada
产品信息如下：
……

请使用完整上架模式输出。
标题要有真实电商平台关键词累加感。
详情文案和 9 图提示词要可直接复制使用。
9 图提示词必须是 9 张独立正方形产品图，不要九宫格、拼图或合成图。
```

## 输出原则

- 标题必须具有平台搜索标题的信息密度
- 详情文案必须是可直接上架的成品
- 9 图提示词必须具备电商信息图属性
- 所有提示词必须独立完整，不能互相引用
- 规则必须跨类目通用，不绑定单一产品类型
- 不得编写用户未提供的参数、等级、背书或绝对化承诺
